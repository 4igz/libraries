MatrixUtil

Simple utility module for working with matrixes. A matrix is simply an array consisting of arrays, e.g:

local matrix = {
	{1, 1, 2},
	{1, 1, 1},
	{3, 3, 3},
}

matrix:Â {{any}},

Searches matrix row wise, and returns a value in a row which matches with the rest of the values of that row. E.g:

local matrix = {
	{1, 1, 1}, 
	{5, 5, 2}, 
	{0, 0, 2},
}

print(MatrixUtil.GetMatchingRowsValue(matrix)) --> 1

Additionally, you can specify depth if you want to control how far the method should check each row. For e.g:

local matrix = {
	{1, 2, 3, 4}, 
	{5, 6, 7, 8}, 
	{1, 1, 1, 0},
}

print(MatrixUtil.GetMatchingRowsValue(matrix, 3)) --> 1 
print(MatrixUtil.GetMatchingRowsValue(matrix, 0)) --> nil
