Inverse Lerp is the inverse operation of the Lerp Node. It can be used to determine what the input to a Lerp was based on its output. For e.g, the value of a Lerp between 0 and 2 with alpha being 1 is 0.5. Therefore the value of an Inverse Lerp between 0 and 2 with alpha being 0.5 is 1.

local num = 2
local goal = 5

num = NumberUtil.inverseLerp(num, goal, 0.7)
print(num) --> -0.43333333333333335
