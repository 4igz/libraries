EnumList

A simple class for creating custom enum lists. An enum list in layman's terms is simply an object used to store enums inside. 

local enumList = EnumList.new("EnumList", {
	PhoneNumber = {
		BabaBoey = 123,
	}
})

print(enumList.PhoneNumber.BabaBoey) --> 123
Generalization iteration!

EnumLists are iterable, e.g:

local enumList = EnumList.new("EnumList", {
	Test = {Alphabet = "A"}
})

for enumName, enum in enumList do
	print(enumName, enum.Alphabet)
end

--> "Test" "A"
note

EnumLists don't provide support for deep-chain enums (even though they're not idiomatic), e.g

local EnumList = require(...)

local enumList = EnumList.new("EnumList", {
	Enum = {
		Deep = {
			MoreDeep = {
				Lol = 5
			}
		}
	}
})

print(enumList.Enum.Deep.MoreDeep.none) --> nil, but won't error..
print(enumList.Enum.Deep.lo) --> nil, but won't error..
print(enumList.Enum.b) --> will error
