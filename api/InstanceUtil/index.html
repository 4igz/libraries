GetInstanceFloorMaterialâ€‹

</>
InstanceUtil.GetInstanceFloorMaterial(
instance:Â BasePart,
raycastParams:Â RaycastParams?,
depth:Â number?
) â†’Â EnumItem

Returns the material the instance is lying on.

  • The 2nd argument can be passed as a RaycastParams object, which will be used in determining the material of instance through ray casting.
  • The 3rd argument can be passed as a number (depth) which will increase the length of the ray by depth studs (on the Y axis). This is only useful when you want to add in more leeway in determining the material of instance reliably, since sometimes the instance may be very slightly over the top of some ground due to it's geometr so in those cases, the ray may not register properly. If this argument isn't specified, then it will default to 0.01.
